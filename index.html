DIY 18650 Lithium Powered Tablet/Computer

A DIY 18650 powered tablet/computer built using an Odroid XU3 Lite single board computer and Odroid VU touch screen.

This uses the same computing and touch screen hardware as the previous prototype but upgraded to lighter 18650 batteries instead of a heavy SLA battery, and a lighter plastic case instead of a heavy timber one.

Additional Specs:

  • Odroid XU3 Lite single board computer running Ubuntu
  • Odroid-VU 9” touch screen display
  • 2S2P Panasonic NCR18650GA 2x3200mAh 8.4v Battery Pack (charged by supplying up to 15v)

Credit
The 2S2P battery pack used in this project is based on the 2S1P battery pack shown here:

DIY Portable Touchscreen Computer

A DIY portable touch screen computer prototype.

  • Odroid XU3 Lite single board computer running Ubuntu
  • Odroid-VU 9” touch screen display
  • Wireless keyboard with touchpad
  • 12v 7ah SLA battery
  • 2x 5a XL4015 Variable Voltage Buck Regulators set to 5v to power the XU3 and VU.
